# 说明
搜索(Minecraft Wiki)[https://minecraft.fandom.com/zh/wiki/Minecraft_Wiki]，使用了API，配合异步HTTP请求，大约消息处理部分能达100条/分钟。

# 需求的库
使用了异步HTTP请求，请安装```httpx```，或根据自己的需求更换为```pip3```、```conda```之类：
```bash
pip install httpx
```
